# Capstone Project – Todoist Testing

**Project Overview**  
This project is part of CMP1979 – Modern Software Development.  
Our team is performing a software testing analysis on the Todoist application.  

Todoist is a task management tool that helps users organize and manage their tasks efficiently.  
The goal of this project is to test core functionalities, identify defects, and improve usability.
